# AI Security System (YOLOv8 + SOLID Pipeline)

Система видеоаналитики в реальном времени, построенная на принципах **SOLID** и объектно-ориентированного проектирования. Проект умеет обнаруживать объекты через камеру и гибко реагировать на события.

## 🚀 Основные фишки
- **Computer Vision:** Использование нейросети YOLOv8 (PyTorch) для детекции объектов.
- **Архитектура:** Полная инверсия зависимостей. Логика детекции отделена от логики действий через интерфейсы (`ABC`).
- **Модульность:** Легко добавить новые детекторы или действия, не меняя основной цикл программы.
- **Действия (Actions):** Автоматическое сохранение снимков при обнаружении человека и отрисовка рамок в реальном времени.

## 🛠 Технологии
- **Python 3.10+**
- **OpenCV** (работа с видеопотоком)
- **Ultralytics YOLOv8** (нейросетевой движок)
- **PyTorch** (Inference моделей)

## 📁 Структура проекта
- `core/` — интерфейсы и базовый Pipeline (управление потоком).
- `detectors/` — реализации детекторов (YOLO, Mock).
- `actions/` — плагины для реакций (сохранение фото, отрисовка).
- `main.py` — точка входа и сборка системы.

##💡 Как это работает (SOLID)
Проект демонстрирует понимание паттернов проектирования:

S (Single Responsibility): Каждый класс отвечает за свою задачу (поиск, отрисовка или сохранение).

O (Open/Closed): Система открыта для новых типов детекторов (например, распознавание лиц), но закрыта для изменения основного пайплайна.

D (Dependency Inversion): Главный цикл зависит от абстракций BaseDetector и BaseAction, а не от конкретных библиотек.
